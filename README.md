## Overview

In this task, I practiced basic Linux commands that are commonly used in DevOps.
This helped me understand how to work with the Linux terminal and manage files, directories, and system resources.

---

## What I Learned

* Navigating directories using terminal
* Creating and managing files and folders
* Editing files using nano
* Managing file permissions
* Monitoring system processes
* Installing and managing packages

---

## Commands Practiced

### File Navigation

* `pwd` → shows current directory
* `ls` → lists files and folders
* `cd` → change directory

### File & Directory Management

* `mkdir` → create directory
* `touch` → create file
* `rm` → delete file

### File Editing

* `nano filename` → edit file

### Permissions

* `chmod` → change file permissions

### System Monitoring

* `top` / `htop` → view running processes

### Package Management

* `sudo yum update`
* `sudo apt install <package>`
